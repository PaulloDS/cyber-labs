# Relatório de Segurança

## 📌 Resumo Executivo

Foi identificado comprometimento do servidor através de vulnerabilidade na aplicação web DVWA.

---

## 🔍 Vulnerabilidades

- Credenciais padrão
- Command Injection

---

## 💀 Exploração

- Acesso via login padrão
- Execução de comandos remotos
- Obtenção de reverse shell

---

## 🚨 Impacto

Controle remoto do servidor.

---

## 🛡️ Recomendações

- Alterar credenciais padrão
- Validar entradas do usuário
- Monitoramento de rede
- Hardening do servidor
